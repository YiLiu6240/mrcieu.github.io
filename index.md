## Highlighted open source software released through GitHub

* [MR-Base](http://www.mrbase.org) enables online Mendelian randomization analysis using a comprehensive manually curated database of GWAS studies. The following MR-Base software is available through GitHub:
    * [TwoSampleMR](https://github.com/MRCIEU/TwoSampleMR) is the core R package for MR-Base
* [PHESANT](https://github.com/MRCIEU/PHESANT) runs a phenome scan (pheWAS, Mendelian randomisation (MR)-pheWAS etc.) in UK Biobank.
* [MELODI](http://melodi.biocompute.org.uk/) is a literature mining platform to identify potential mechanistic pathways between exposures and disease outcomes - [source code](https://github.com/MRCIEU/melodi)
* [Meffil](https://github.com/perishky/meffil) Efficient algorithms for analyzing DNA methylation data
